# Roadmap

## Introdução

O Zelon é uma plataforma em constante evolução.

Nosso objetivo não é apenas adicionar funcionalidades.

Estamos construindo um ecossistema completo para ajudar pessoas a organizar suas finanças, construir patrimônio e alcançar independência financeira.

Este roadmap representa a direção estratégica do produto.

As prioridades podem evoluir conforme aprendemos com usuários, dados e novas oportunidades.

# Nossa Ambição

Não queremos construir apenas mais um aplicativo financeiro.

Queremos construir a principal plataforma de evolução financeira pessoal da América Latina.

Uma plataforma onde qualquer pessoa possa:

- Organizar
- Planejar
- Investir
- Evoluir
- Construir patrimônio
- Alcançar independência financeira

Tudo em um único ecossistema.
