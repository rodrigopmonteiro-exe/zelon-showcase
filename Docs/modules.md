# Modules

## Visão Geral

O Zelon foi construído como uma plataforma modular.

Cada módulo possui uma responsabilidade específica dentro da jornada financeira do usuário.

Separadamente, eles resolvem problemas pontuais.

Juntos, eles criam um sistema integrado de evolução financeira.

---

# A Jornada do Usuário

A arquitetura do produto foi desenhada para acompanhar o usuário desde os primeiros registros financeiros até o planejamento de longo prazo.

```text
Capture
   ↓
Meu Mês
   ↓
Patrimônio
   ↓
Planejamento
   ↓
Estratégias
   ↓
Evolução
   ↓
Simulador
   ↓
Alliance
```

Cada módulo alimenta o próximo.

---

# Capture

## Objetivo

Reduzir a fricção da entrada de dados.

A maioria dos aplicativos financeiros falha porque exige esforço excessivo para registrar despesas.

O Capture foi criado para simplificar esse processo.

### Funcionalidades

- Registro rápido de despesas
- Categorização automática
- Integração com o Zelon
- Histórico de lançamentos

### Exemplo

```text
25 Uber
48 Padaria
120 Mercado
```

O sistema interpreta automaticamente o lançamento.

### Benefício

Menos tempo registrando.

Mais tempo tomando decisões.

# Alliance

## Objetivo

Aumentar engajamento e consistência.

### Conceito

Alliance é a camada de gamificação do Zelon.

Ela transforma progresso financeiro em progresso visível.

### Componentes

#### XP

Experiência obtida através de ações positivas.

#### Coins

Moeda interna da plataforma.

#### Níveis

Sistema de progressão.

#### Missões

Desafios para incentivar hábitos.

#### Loja

Resgate de recompensas.

#### Ranking

Comparação saudável entre participantes.

#### Clube Lenda

Reconhecimento dos usuários mais consistentes.

### Benefício

Cria motivação contínua para manter hábitos financeiros saudáveis.

# Como Tudo se Conecta

O maior diferencial do Zelon não está em um módulo específico.

Está na integração entre eles.

Um simples registro realizado no Capture pode impactar:

```text
Capture
   ↓
Meu Mês
   ↓
Planejamento
   ↓
Estratégias
   ↓
Evolução
   ↓
Alliance
```

Isso transforma dados financeiros em inteligência, progresso e motivação.

# Filosofia dos Módulos

Cada módulo foi criado para responder uma pergunta fundamental.

| Módulo | Pergunta |
|---------|-----------|
| Capture | O que aconteceu? |
| Meu Mês | Como está meu mês? |
| Patrimônio | O que eu possuo? |
| Dívidas | O que preciso resolver? |
| Reserva | Estou protegido? |
| Planejamento | O que devo fazer? |
| Estratégias | Como posso melhorar? |
| Evolução | Estou progredindo? |
| Simulador | Para onde estou indo? |
| Alliance | Como manter a consistência? |

# Conclusão

O Zelon não foi projetado como um aplicativo de controle financeiro.

Foi projetado como um sistema de evolução financeira.

Cada módulo representa uma etapa dessa jornada.

Juntos, eles ajudam usuários a transformar pequenas decisões diárias em resultados financeiros duradouros.
